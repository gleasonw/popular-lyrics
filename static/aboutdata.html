About The Data

Here is all of the information on the dataset we used

What is the data?

The dataset compliles the year end Billboard top-100 song list from the years 1965 to 2015

Where did this data come from?

The data was scraped from wikipedia.com and a few lyric sites like metrolyrics.com.

Who created the dataset?

The dataset was originally created and uploaded to Github by Kaylin Pavlik and again published to Kaggle by RakanNimer.

Can I use the data too?

Yes! it is cc from https://github.com/walkerkq/musiclyrics
